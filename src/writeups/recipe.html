Writeup

The recipe box is done with the React JavaScript library. It can be used to store recipes. One box holds the recipe name, the second holds the recipe ingredients, and the last holds the recipe instructions. This uses the localstorage API to persist data in your browser so it is not lost. Since this is a react app, it uses Webpack for bundling and Babel for transpilation

There are different prperties it needs to keep track of: The recipes themselves, the recipe, name, ingredients and instructions used store keystrokes when need, a label which is used in the modal, two booleans, one for editting, and the other for the modal, and finally, a details array that will be sent to the display. There are methods for adding, editing, deleting, and saving recipes. A lifecycle method is used for saving. It also uses the React modal library.

Starting from the create recipe button this is the logic

  • 1. It will check the text content of the button. If it is "create"
  • 2. State is set where the label is set to create and the three: name, ingredients and instructions are set to empty strings in case they are full
  • 3. If the label is edit, it will populate the three proeprties with what is stored in the details array
  • 4. The modal is opened and it has a form for the name, ingredients and instructions. There is an instruction to separate each new ingredient and/instruction with a newline
  • 5. Upon submission, preventDefault is called to prevent page refresh
  • 6. the contens of the ingredients and instructions has the split method called using regular expression to separate content for each item in the two arrays.
  • 7. An id is generated.
  • 8. State is set for the recipes array using the id, the name, and both arrays

For the first box that is the recipe list, it uses the recipes array and the map method to loop through and display each name and uses id for the keys. There is a function passed into the recipe list to handle clicking.

When is recipe from the list is clicked on

  • 1. It will take the id of the recipe
  • 2. It will use findIndex method on the recipes array using the id
  • 3. It will find the instructions, ingredients and name of the recipe using the index on the same array
  • 4. State is set for the details array with the above information.

Since the details array has been filled, it will display the data in the ingredients and instructions boxes using the map method. It uses a ternary operator to make sure there is data at all. The delete and edit buttons are also dispalyed here which also uses a ternary to check for data existence.
