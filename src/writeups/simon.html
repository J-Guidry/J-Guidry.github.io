Writeup

Simon game. Click the start button and then it will play a sound and highlight a color. Click the color and it will repeat again with a new color added. Click the sequence and do this for 20 turns and you win. There is a strict mode button which will start all over if you get a color wrong.

It started with brainstorming the logic. If there was no UI, how would it go on paper? For this game, it needs to keep track of certain things. The player's input, the randomized sequence of colors, whether or not strict mode is on, the turn count, whether or not it is your turn, whether or not the buttons are diabled, and a group of the colors to choose from themselves. The player's input and the turn count can be a counter that increments when the right condition is met. The group of colors and the sequence can be two JavaScript arrays, the second one being added to when a condition is met. Strict mode can be a boolean. This uses React js to keep track of all the above as state.
